# Title

整合船舶尺寸與座標估算、船名OCR識別以及特徵比對的綜合分析

## Install

```
pip install requirement.txt
```

## Usage

```
__ship_feature__
experiment.py -> 實驗程式碼
main.py -> 船舶外觀特徵體取＆比對

ship_name
1. data_cleaning.py -> 資料清洗
2. ship_detection.py -> 船舶識別
3. ocr.py -> 文字識別

ship_size
experiment.py -> 實驗程式碼
main.py -> 船舶長寬及經緯度估算
```
