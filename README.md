# Title

整合船舶尺寸與座標估算、船名OCR識別以及特徵比對的綜合分析

## Install

```
pip install requirement.txt
```

## Usage

**ship_feature**

* experiment.py -> 實驗程式碼

* main.py -> 船舶外觀特徵提取＆比對

**ship_name**

* data_cleaning.py -> 資料清洗

* ship_detection.py -> 船舶識別

* ocr.py -> 文字識別

**ship_size**

* experiment.py -> 實驗程式碼

* main.py -> 船舶長寬及經緯度估算
